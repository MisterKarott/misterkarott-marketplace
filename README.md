# MisterKarott Plugins Marketplace

Plugins Claude Code par Chris (MisterK).

## Installation

```bash
claude plugin marketplace add https://github.com/MisterKarott/misterkarott-marketplace
```

## Plugins disponibles

### glm-quota
Z.ai/GLM quota statusline, detailed quota skill, and MCP coherence check for Claude Code.

```bash
claude plugin install glm-quota
```

### claude-quota
Claude Pro quota statusline and detailed quota skill for Claude Code.

```bash
claude plugin install claude-quota
```

### stratavarious
Persistent memory system for Claude Code — replaces /handoff.

```bash
claude plugin install stratavarious
```
